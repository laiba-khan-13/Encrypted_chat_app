# SecureChat - Encrypted Chat Application

A production-grade, multi-client encrypted chat application built with Python for the SyntecxHub Cybersecurity Internship.

## Features

- **AES-128-CBC Encryption** via Fernet (symmetric encryption)
- **Multi-Client Support** with threading
- **Private Messaging** (Whisper)
- **Admin Controls** - Kick, Ban, Mute, Unmute, Broadcast
- **Rate Limiting** - Spam protection
- **Typing Indicators**
- **Message History**
- **Heartbeat Keep-Alive**
- **Structured Logging**
